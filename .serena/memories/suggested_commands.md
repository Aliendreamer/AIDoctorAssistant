# Suggested Commands

> NOTE: Project is greenfield. Update this file once the solution structure is created.

## .NET / dotnet
- `dotnet build` — build the solution
- `dotnet test` — run tests
- `dotnet run --project <path>` — run a specific project
- `dotnet format` — format code
- `dotnet restore` — restore NuGet packages

## Git
- `git status`, `git log`, `git diff` — standard inspection
- `git add <files> && git commit -m "..."` — commit

## OpenSpec workflow
- `/opsx:explore` — explore/think through ideas
- `/opsx:propose` — create a new change proposal with specs and tasks
- `/opsx:apply` — implement tasks from a change
- `/opsx:archive` — archive completed change
